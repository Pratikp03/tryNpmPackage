# What is this? 

Get shadow to buttons.

# Installation

 `npm i shadowToAll --save`


 Then.. 


 ```
 import {shadowToAll} from 'shadowToAll';

 shadowToAll({
     shadow_Type:'soft',
     padding: 'true',
     imgColor: 'true'
 })
 ```

 ## Options

 * *shadow_Type* - _hard | soft_ (Defaults to soft)
 * *padding* - _boolean (Defaults to false)
 * *imgColor* - _boolean (Defaults to false)
